6 · Non-obvious design decisions

FOUC-safe theme toggle

Inline <script is:inline> in <head> reads localStorage.theme and applies .dark to <html> before first paint. ThemeToggle.svelte is the only island using client:load; every other island is client:visible or client:idle. Tested by theme.spec.ts#no-FOUC, which pre-seeds localStorage then asserts the initial className already contains the saved theme.

LCP never JS-dependent

TypingHero renders its initial phrase as static HTML. The Svelte island (client:idle) only animates subsequent phrases. If JS never loads — slow network, crawler, no-JS user — the largest contentful paint element is already final.

MetricCounter pre-renders truth

The final metric value is in the raw DOM. Animation counts up to it via IntersectionObserver; it never creates the number from JS. Crawlers, no-JS users, and prefers-reduced-motion: reduce users see the same truth. homepage.spec.ts greps the raw DOM for ['67%','93%','40%','71.3%','70%','0.87+'].

Conditional KaTeX CSS

NoteLayout.astro inspects entry.data.requiresMath; only math-bearing notes emit <link rel="stylesheet" href="/katex.min.css">. Homepage and non-math notes save ~25 KB gz. notes.spec.ts asserts the link count is 0 on non-math pages and > 0 on math pages.

Strict exact-pinning

No ^, no ~, no ranges, no tags. .npmrc sets save-exact=true. scripts/check-dep-pinning.mjs walks every dep block and runs semver.valid() — anything other than an exact version fails the hook and fails CI. Dependabot bumps exact pins in lockstep via versioning-strategy: increase.

7 · Known gaps

Header-level CSP is deferred. GitHub Pages cannot set HTTP headers; the CSP ships as <meta http-equiv>. The stronger header-level CSP, Permissions-Policy, and HSTS are documented as a Cloudflare Transform Rule in README.md. Follow-up: apply the rule once the domain sits behind Cloudflare (dashboard access required, outside this PR's scope).
Per-note OG images not automated. Default OG image ships at /og/og-default.svg. The scripts/generate-og.ts satori pipeline is wired but not invoked per-note in the build graph. Acceptable for launch — the default image carries the headline and accent bar.
Streak widget embeds an external image. streak-stats.demolab.com is a third-party redirector; the CSP allowlist accommodates it. Follow-up: build-time snapshot to remove the external dependency.

8 · Why approve

Every claim in this doc has a corresponding automated gate. make verify locally runs the exact checks GitHub Actions runs on push. The deploy job does not fire until every upstream gate — lint, type-check, unit tests, build, e2e on three browsers, Lighthouse CI with budget assertions, link check, size-limit — has passed. Rollback is a single revert: no schema, no data, no stateful surface.

The mentor review at specs/reviews/pr-11-mentor-review.md probes the design decisions above with Socratic intensity and comes out at "Approve with follow-ups" — none of which block merge.

This PR earns its "atomic cut-over" framing. The v1 site was a placeholder; there is no partial-migration path that isn't worse than landing v2 whole. Every budget is within spec, every route is reachable, every data structure is Zod-validated, and every hook — git, CI, pre-deploy — is wired before merge rather than after.
